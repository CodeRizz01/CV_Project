# CV_Project
## Face Detection using OpenCV

### Overview

This project implements a real‑time face detection system using OpenCV and Python. It captures video from a webcam and detects human faces.

---

## Features

* Real‑time face detection
* Multiple face detection
* Lightweight implementation
* Easy to run

---

## Installation

### Install Requirements

```bash
pip install opencv-python
```

---

## Usage

Run the following command:

```bash
python cv.py
```

Press **q** to exit.

---

## Project Structure

```
face-detection/

├── face_detection.py

```

## Output

The webcam will open and faces will be detected in real time.
